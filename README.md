# Ameen Ambalakuth — Portfolio

A single-file, dark & techy (monochrome) personal portfolio website.
No build step, no dependencies — just open it in any browser.

## Files
- `index.html`  — the entire website (HTML + CSS + JS in one file)
- `profile.jpg` — (optional) add your photo here to show it in the hero

## Run locally
Double-click `index.html`, or drag it into any web browser.

## Add your photo
1. Put your image in this folder named `profile.jpg`
2. In `index.html`, find this line in the hero avatar:
   `<span class="mono">AA</span>`
   and replace it with:
   `<img src="profile.jpg" alt="Ameen Ambalakuth" />`

## Publish for free (get a live link)
- GitHub Pages: create a repo, upload these files, enable Pages in Settings.
- Netlify / Vercel: drag this folder onto their dashboard — done.
